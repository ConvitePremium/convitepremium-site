# Convite Premium

Site oficial do Convite Premium.

## Como testar
Abra o arquivo `index.html`.

## Vídeos
Coloque na pasta `videos/`:

- `carrossel-encantado.mp4`
- `mundo-doce.mp4`
- `fazendinha-rosa.mp4`

## Logo
Troque em `assets/logo/logo.png`.

## Catálogo
Os modelos ficam em `data/convites.json`.

## Domínio
Configurado para `www.convitepremium.shop` no arquivo `CNAME`.
